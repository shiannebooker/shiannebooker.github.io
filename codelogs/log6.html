I have decided to research the CSS property 'display:'

The websites I consulted are:

  • Modzilla Developer
  • Geeks for Geeks
  • Programiz
  • Log Rocket Blog

The main difference in the information I found on each site is how in depth they go about the display property. I would say Log Rocket had the most information, While Geeks for Geeks & Programiz had the least. Although they all meet the requirments to explaing what it is and how to use it. They all have similar structure. One thing that stood out to me was Geeks for Geeks page. They have an interactive window that lets you press a button that correlates to the type of display, and blocks will move to show you where your content will move. Modzilla's website talked about what browsers support which displays which I thought was super helpful. Overall I think they all do a good job of relaying the same information in their own way.

I think that a code reference should bare minimum include what the code is and if there are different types, what is does funtion-wise, and example code.

I feel like I can't necessarily say one is bad, because they all do a great job explaining the property, but I do think that Geeks for Geeks is my favorite because the interactive display makes it so simple to figure out which property you need to use for whatever you are looking to code and design.
